# Library-System

A simple database management system for handling library records.

## Features
 -	View, add and delete books.
 -	Manage library members.
 -	Issue and return books.
 -	Manage fines for overdue books.
 -	Manage reservations of books.

## Entities
-	Books
-	Members
-	Loans
-	Fines
-	Reservations
-	Categories
-	Publishers
